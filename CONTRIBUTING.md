# Contributing

## Setup

```bash
composer install
```

## Run Tests

```bash
vendor/bin/phpunit
```

## Code Style

- Target PHP 8.4+
- Prefer strong typing and small focused classes
- Keep public APIs explicit and predictable
- Avoid framework-specific dependencies in the core package

## Pull Requests

- Keep changes focused
- Add or update tests when behavior changes
- Update `README.md` when the public API changes
- Update `CHANGELOG.md` for user-visible changes
