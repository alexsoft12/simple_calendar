# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 2026-03-24

### Added

- Framework-agnostic calendar core for PHP 8.4+
- Fluent builder API for rendering and configuration
- Translation support through PHP language files
- Bootstrap 4 friendly default template
- Configurable navigation, templates and event rendering
- PHPUnit test suite for core behavior

### Security

- Escaped event attribute values before rendering them into HTML attributes
